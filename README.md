# GoddessGPT - AI Assistant for Women

A modern, responsive web application built with Next.js and TypeScript, designed as an AI chatbot assistant for women seeking solutions in healthcare, legal, and mental wellness domains.

## Getting Started

1. Install dependencies

```bash
pnpm install # or npm install / yarn install
```

2. Run the development server

```bash
pnpm dev # or npm run dev / yarn dev
```

3. Open http://localhost:3000 to see the landing page.

## Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui-style primitives (Button)
- lucide-react icons

## Notes
- The "Sign in with Google" button is a mock CTA.
- Theme tokens are set in `app/globals.css` for easy customization.
